# Post Format

keys work like this:

```mdx
---
key: value
---

content
```

keys:

- title: string                               | Name Of Article
- author: string                              | Name Of Author
- date: string                                | Date Published (YYYY-MM-DD)
- published: boolean (as bool or string type) | Whether the post is published or not
- categories: string[]                        | Categories Of Article, For Sorting
- tags: string[]                              | Tags Of Article, For Sharing And Hashtags
- heading_image: string                       | Image Shown At Article At Top In Its Card

categories:
